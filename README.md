# EXCEL-PROJECT

# 📊 Amazon Sales Analysis Dashboard | Microsoft Excel

## 📌 Project Overview

This project showcases an interactive sales analysis dashboard built in Microsoft Excel using advanced analytical features. The dashboard transforms raw sales data into meaningful business insights through Pivot Tables, Pivot Charts, KPIs, and Excel functions, enabling stakeholders to monitor business performance and make data-driven decisions.

The project demonstrates practical Excel skills required for Business Analyst and Data Analyst roles.

---

## 🛠️ Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Timeline Filter
* Excel Functions
* Conditional Formatting

---

## 📂 Dataset

The dataset contains Amazon sales transaction records, including:

* Order ID
* Order Date
* Customer Name
* Product Category
* Product Name
* Region
* Sales
* Quantity
* Profit
* Discount

---

## 🎯 Project Objectives

* Analyze overall sales performance.
* Monitor profitability across products and regions.
* Identify top-performing product categories.
* Track sales trends over time.
* Build an interactive dashboard for business reporting.

---

## 📊 Dashboard Features

The dashboard includes:

* KPI Cards (Total Sales, Total Profit, Total Orders, Average Order Value)
* Sales Trend Analysis
* Profit Analysis
* Region-wise Sales Performance
* Category-wise Sales Analysis
* Top Performing Products
* Interactive Time Line Filter
* Slicers
* Dynamic Charts and Pivot Tables

---

## 📌 Key Insights

* Technology products generated the highest revenue.
* A small number of products contributed significantly to total sales.
* Sales varied across different regions, highlighting growth opportunities.
* Monthly sales trends revealed seasonal demand patterns.
* Profitability differed by product category and discount levels.

---

## 💡 Business Recommendations

1. Focus marketing efforts on high-performing product categories to maximize revenue.
2. Review discount strategies for low-profit products to improve overall profitability.
3. Increase inventory for consistently high-demand products to prevent stock shortages.
4. Develop targeted campaigns for underperforming regions to increase sales.
5. Use seasonal sales trends to improve inventory planning and sales forecasting.
